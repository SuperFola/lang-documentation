From the source

Needed packages

  • CMake
  • Python >= 3.6 (to run the tests)
  • Java (to build the lexer/parser if you want to modify the grammar)

Procedure

user:~$ git clone -j$(nproc) --recursive git@github.com:Kafe-lang/Kafe.git
user:~$ cd Kafe && mkdir build && cd build
user:~/Kafe/build/$ cmake ..
user:~/Kafe/build/$ make -j$(nproc) install

Building the lexer/parser

Note: Do it only if you changed the grammar
user:~/Kafe/$ java -jar antlr/antlr-4.7.1-complete.jar Kafe.g4 \
    -Dlanguage=Cpp -no-listener -visitor -o include/generated

Running the tests

user:~$ git clone -j$(nproc) --recursive git@github.com:Kafe-lang/Kafe.git
user:~$ cd Kafe && mkdir build && cd build
user:~/Kafe/build/$ cmake .. -DTests=True
user:~/Kafe/build/$ make -j$(nproc) && make KafeTests
